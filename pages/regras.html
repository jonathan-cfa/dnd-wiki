INICIO RAÇAS CLASSES ANTECEDENTES MAIS

📜 SISTEMA DE REGRAS BÁSICAS

🎲 Como rolar dados

Base do sistema
  • O dado principal é o d20
  • Outros dados: d4, d6, d8, d10, d12
  • Quase toda ação usa 1d20 + modificadores

🧠 Testes de habilidade

Como funciona
  • 1d20 + atributo (Força, Destreza, etc.)
  • + bônus de proficiência se tiver
  • Comparado com uma CD (dificuldade)

⚖️ Vantagem e desvantagem

Regra simples
  • Vantagem: rola 2d20 e usa o maior
  • Desvantagem: rola 2d20 e usa o menor

🎯 Classe de Dificuldade (CD)

O que é CD
  • Número alvo para sucesso
  • Ex: CD 15 = precisa tirar 15 ou mais
  • Definida pelo mestre

😴 Descanso

Descanso curto
  • Duração: cerca de 1 hora
  • Recupera habilidades limitadas
Descanso longo
  • Duração: cerca de 8 horas
  • Recupera vida e magias
  • Restaura recursos principais

📌 Resumo rápido

  • d20 resolve quase tudo
  • Atributos modificam resultados
  • CD define dificuldade
  • Vantagem muda probabilidade
  • Descanso recupera recursos
